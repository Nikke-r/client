# client
Mobile Client 

This app is still under development.

To run the app you need the following applications installed on your machine/mobile phone:

<ul>
  <li><a href='https://git-scm.com/downloads'>Git</a></li>
  <li>npm (<a href='https://nodejs.org/en/'>NodeJS</a>)</li>
  <li><a href='https://expo.io/'>expo cli</a> (can be installed later)</li>
  <li>expo client (for mobile) (<a href='https://play.google.com/store/apps/details?id=host.exp.exponent&hl=fi'>Adroid</a> | <a href='https://apps.apple.com/us/app/expo-client/id982107779'>iOS</a>)</li>
</ul>

Steps to run the app:

1. Clone the repository to your machine using Git (git clone https://github.com/Nikke-r/client.git)
2. Run 'npm install' inside the project folder
3. Run 'npm start' (If you don't have expo cli installed it will ask you to install it, select 'Y')
4. Scan the QR code using a camera app if on iOS or the expo client if on Android
